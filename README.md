# DUNE Launcher Map Data

Этот репозиторий читает кнопка `Подгрузить карту DUNE` в DUNE Launcher.

Клади сюда клиентские файлы карты SCUM, которые должны попасть в `SCUM\Content\Paks\~mods`.

Поддерживаются файлы:
- `.pak`
- `.ucas`
- `.utoc`
- `.sig`

Можно класть файлы прямо в корень репозитория или в подпапки. Лаунчер сохранит структуру подпапок внутри `~mods`.
